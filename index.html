The basics

This script uses the following markup to trigger slideshow behavior:


				<div class="slideshow">
					<div>Slide 1</div>
					<div>Slide 2</div>
					<div>Slide 3</div>
					<div>Slide 4</div>
					<div>Slide 5</div>
				</div>

As you can see, a simple .slideshow class with some divs in it suffices (with the default css).

You can use a different container element or slides if you like. You'll just need to change some of the default css)

The options

The slideshow only supports a handful of options via attributes on the .slideshow element:

data-height [default: false]
if set to "individual" triggers per-slide height of the slideshow. Not recommended but forces the slideshow to resize every slide.
data-delay (in milliseconds) [default: 0]
delays the start of the slideshow
data-timeout (in milliseconds) [default: 5000]
The time a slide is visible. Keep in mind the transition from one slide to another eats into this

The right slideshow above uses the following code:


				<div class="slideshow"
					data-height="individual"
					data-delay="2000"
					data-timeout="7000">
					<div>Slide 1</div>
					<div>Slide 2</div>
					<div>Slide 3</div>
					<div>Slide 4</div>
					<div>Slide 5</div>
				</div>
